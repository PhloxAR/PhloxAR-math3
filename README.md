PhloxAR-math3
====

Provides 3D mathematical functions for PhloxAR.

Features:
  * Object Oriented and Procedural interfaces
  * Matrix (3x3, 4x4)
  * Quaternion
  * Vector (3D, 4D)
  * Plane
  * Ray
  * Line / Line Segment (3D)
  * Rectangle (2D)
  * Axis Aligned Bounding Box (AABB / AAMBB)
  * Geometric collision / intersection testing

Requirements
------------
Numpy
multipledispatch
